# Arbres — feuillage des cyprès (la partie verte)

**Surface visée :** le feuillage des cyprès (aujourd'hui un dégradé de verts
calculé). Le tronc, lui, reste en bois.

Dépose ici **3 fichiers** :

- `diff`   — couleur (Albedo / Diffuse)
- `nor_gl` — relief (Normal **GL**)
- `rough`  — mat / brillant (Roughness)

**Conseils :** 1K–2K, JPG/WebP. Idéal : **feuillage dense / haie / lierre /
aiguilles de conifère**, d'un vert profond.

> ⚠️ Cas particulier : plaquer une texture plate sur un arbre en 3D est
> délicat. Prévois que je devrai sans doute **ajuster le tuilage et/ou la
> teinte** pour un beau rendu. Une texture « haie » ou « buisson » marche
> beaucoup mieux qu'une feuille isolée.

# Ruines / pierres éparses

**Surface visée :** les pierres et éclats à demi enfouis dans le sol autour du
temple (aujourd'hui une couleur unie gris-brun, sans texture — `ScatterStones`).

> Les **gros vestiges** (tambours de colonnes, chapiteaux brisés) gardent le
> marbre veiné : une texture de blocs/rochers irait mal sur un fût rond.

Dépose ici **3 fichiers** :

- `diff`   — couleur (Albedo / Diffuse)
- `nor_gl` — relief (Normal **GL**)
- `rough`  — mat / brillant (Roughness)

**Conseils :** 1K–2K, JPG/WebP. Idéal : roche / rocher patiné, pierre brute ou
calcaire usé, dans un ton clair et chaud pour rester dans l'esprit du temple.

# Sol extérieur — parvis sableux

**Surface visée :** le grand sol devant le temple (aujourd'hui une texture de
sable générée par calcul).

Dépose ici **3 fichiers** :

- `diff`   — couleur (Albedo / Diffuse)
- `nor_gl` — relief (Normal **GL**)
- `rough`  — mat / brillant (Roughness)

**Conseils :** 1K–2K, JPG/WebP. Idéal : sable, terre tassée, gravier ou
grandes dalles de pierre.

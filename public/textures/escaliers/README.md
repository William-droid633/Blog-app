# Escaliers

**Surface visée :** le grand escalier d'accès au temple (aujourd'hui en marbre
généré par calcul).

Dépose ici **3 fichiers** :

- `diff`   — couleur (Albedo / Diffuse)
- `nor_gl` — relief (Normal **GL**)
- `rough`  — mat / brillant (Roughness)

**Conseils :** 1K–2K, JPG/WebP. Idéal : marbre, pierre claire ou dalle.

> Si tu veux le même rendu que les murs, mets-y les mêmes fichiers que
> `murs-facade/` (ou dis-moi de réutiliser ceux des murs).
